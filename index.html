GoSemanticFuzz

GoSemanticFuzz is a semantic fuzzer for Go code. It generates random programs that are semantically correct according to a set of rules specified by the user. Check out the project here.

SockChat

SockChat is a real-time chat application built using Node.js, Express, and Socket.io. It allows users to join different chat rooms and communicate with each other in real-time. Check out the project here.

GoAISlackBot

GoAISlackBot is a chatbot built with Go and Slack's API. It can respond to user messages and perform tasks like creating new channels, inviting new members, and sending messages. Check out the project here.
